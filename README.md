# Overview

This project is a Python arcade-style game designed to help me improve my skills in game development. My goal was to create a multi-level game with increasing difficulty while practicing proper use of sprites, collision detection, and sound effects.

In this game, the player controls a character that must collect coins while avoiding moving enemies. Each level introduces more coins, faster enemies, and more enemies, creating a progressively challenging experience. The player uses the arrow keys to move, and a 3-second countdown occurs at the start of each level before the played can move and before enemies can harm the player. After collecting all coins or losing to an enemy, the game displays a message and allows restarting by pressing the enter key .

The purpose of writing this software was to combine game mechanics with real-time user input handling, timed events, and dynamic sprite management. It is a personal project intended to explore the capabilities of the Python Arcade library and gain hands-on experience with game loops and state management.
